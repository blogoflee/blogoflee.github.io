Hugoをインストール

1.バイナリのインストール（推奨：シンプルで高速）

Hugo Releases で対応するオペレーティングシステムバージョンのHugoバイナリファイル（hugoまたはHugo.exe)をダウンロード

MacOSならHomebrewでインストール

brew install hugo

２.ソースコードのインストール

まず依存ツールをインストール：

  • Git
  • Go 1.3+ (Go 1.4+ on Windows)

GOPATH環境変数を設定し、ソースコードを取得してコンパイルします

$ export GOPATH=$HOME/go

$ go install github.com/gohugoio/hugo@latest

サイトの生成

例えば、/path/to/site パスを生成したい場合は、Hugoを使用してサイトをすばやく生成します

$ hugo new site /path/to/site

このようにして、/path/to/siteディレクトリに初期サイトが生成されます。

$ cd　/path/to/site

サイトのディレクトリ構造：

▸ archetypes/

▸ content/

▸ layouts/

▸ static/

　config.toml

ブログを作る

$ hugo new post/first.md

post/first.mdをオッペンする、draft=trueを削除、編集：

---
date: "2023-10-25T08:36:54-07:00"
title: "first"
 
---

### Hello Hugo

 1. aaa
 1. bbb
 1. ccc

themeをインストール

themeリストで好きなthemを選んで、アドレスをコピーし

$ cd /path/to/site/themes

$ git clone https://github.com/spf13/hyde.git

Hugoを実行

$ cd /path/to/site

$ hugo server --theme=hyde --buildDrafts

ブラウザで: http://localhost:1313

デプロイ

GitHub Pagesにデプロイしたら、まずGitHub上にリポジトリを作成し、username.github.ioという名前をつけます（usernameをGitHubユーザー名に書き換えます）

$ cd /path/to/site

$ hugo --theme=hyde -b="https://username.github.io/""

（上記のコマンドでは下書きページは生成されないことに注意してください。記事が生成されない場合は、記事のヘッダーからdraft=trueを削除して再生成してください）

すべてがうまくいけば、すべての静的ページがpublicディレクトリに生成され、public内のすべてのファイルが新しく作成されたリポジトリのmasterブランチにプッシュされます。

`$ cd public

$ git init

$ git remote add origin https://github.com/username/username.github.io.git

$ git add -A

$ git commit -m “first commit”

$ git push -u origin master`

http://username.github.io/ にアクセスしてみましょう。
