Kubectlコマンド

Tue, Jul 30, 2024

podsを取得

kubetctl get pods | grep keyWord

deployを取得

kubetctl get deploy | grep keyWord

pod を削除

kubetctl delete pod podName -c containerName

deploymentを削除

kubetctl delete deployment name

config properties file nameをチェック

kubetctl get cm | grep keyWord

configをチェック

kubetctl describe cm name

config properties fileを編集

kubetctl edit cm fileName

kubernetesのイベント監視

kubetctl get events | grep name

domainをチェック

kubetctl get vs

pod数を拡大

kubetctl scale deployment –replicas=number nameSpace

pod ログをチェック

kubetctl logs -f podName -c containerName kubetctl logs --tail=50 podName -c containerName kubetctl logs --since=1h podName -c containerName

podのログ、設定などをチェック

kubetctl exec -it podName -c containerName --sh
