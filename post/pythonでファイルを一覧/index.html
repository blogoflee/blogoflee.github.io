Pythonでファイルを一覧

Wed, Nov 8, 2023

結論

import glob
from natsort import natsorted

def get_files(path):
    """
    ファイルリストを取得
    Args:
        path: 
         例：/user/grade.xlsx
    Returns:
      files list
    """
    files = natsorted(glob.glob(path + "*.xlsx"))
    return files

pip install natsort

内容

ディレクトリのファリル名をsortして表示することを考えます。

glob

ファイル名の取得はglob.globで行うことができます。

入力は*.xlsxのような正規表現でも扱えます。

natsort

natsortを使うと、文字列内の数字を認識してsortしてくれます。
